## About the Project

This project is a **3D sci-fi soulslike demo** developed in Unity.

The demo focuses on core gameplay systems inspired by the soulslike genre, set in a futuristic science-fiction environment.

### Features
- Melee combat system
- Lock-on camera
- Player and enemy HUD
- In-game UI with menus
- Sound effects integration
- Modular and expandable gameplay architecture

This project is intended as a **technical and gameplay prototype**, showcasing core mechanics and systems rather than a complete game.
